Projects

  • Rodent Baiting in New York City (NYC) - Using Hadoop

  • Abstract
    The aim of this analytic project was to find the factor which can be best used to predict the occurrence of Rodents of a particular area in New York City (NYC). Using few of "311 NYC complaint" databases we tried to find which factor (complaint) causes an increase in the number of rodent complaints in each area of NYC .We also analyzed the frequency of rodent complaints made in the city with respect to the temperature. Furthermore, we used the 311 rodent complaints database to try and get a rough estimate of the rodent population of the city. All this was done using Hadoop MapReduce, Apache Pig, Hive.
